Technical Skills

I first started learning Html and Css on FreeCodeCamp in late 2021. I then proceeded to learn Javascript via FreeCodeCamp and Eloquent Javascript by Marijn Haverbeke. Once I had a basic understanding of Javascript I began to use FreeCodeCamp to learn React as well as Redux. I have since created a number of projects using both vanilla Javascript, React and Redux which you can check out in the projects section. I have recently learnt about D3, a javscript library for dynamic data visualisation, and SVG to allow for greater data visualistion and customability in my projects. I am currently studying Bash and PostgreSQL to further my knowledge of relational databases. I have used source control features such as Git and Github for a number of my latest projects and have become comfortable in using it to ensure a clear history of any changes in the code. In the future I would love to start learning another language, most likely Python.

Drum Machine App

A Drum Machine app created with React and Redux that allows the user to create their own drum beat. It allows the user to either click or use the keyboard to create the sounds as well as having a toggle buton to allow the user to switch to a different sound set.

25 + 5 Clock App

A 25 + 5 Alarm Clock app which counts down from the users chosen time. Once the time runs out an alarm sounds and the break timer starts. The alarm will sound again once this timer runs out at which point the clock revert back to previous time length and continue to count down. This is based of the Pomodoro work method which is meant to improve productivity and reduce procrastination. This method involves working for roughly 25 minutes before taking a short 5 minute break.
